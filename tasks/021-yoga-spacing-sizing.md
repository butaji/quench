# Task 021: Yoga Spacing & Sizing

## Goal
Map padding, margin, gap, width, height, and min/max dimensions to Yoga.

## Acceptance Criteria
- [ ] `padding` / `paddingTop` / `paddingX` / `paddingY` → `YGEdge`.
- [ ] `margin` / `marginTop` / `marginX` / `marginY` → `YGEdge`.
- [ ] `gap` / `gapColumn` / `gapRow` → `YGGutter`.
- [ ] `width` / `height` (`number`, `percent string`, `auto`) → `YGValue`.
- [ ] `minWidth`, `maxWidth`, `minHeight`, `maxHeight` mapped.
- [ ] Unit test: complex spacing config produces expected Yoga layout.
